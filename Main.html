Xbox One

  • Released: November 22, 2013
  • Developer: Microsoft
  • Most powerful console: Xbox One X
  • CPU: Custom 2.3 GHz AMD 8-core APU
  • Memory: 12GB GDDR5
  • Graphics: 1.172 GHz AMD Radeon GCN
  • GFLOPS:6 Teraflops

The Xbox One currently is the most powerful console to date. specifically the Xbox One X. This console is generating a whopping 6 Teraflops of graphical computing power, with 12GB of GDDR5 Memory. The only catch with this kind of system, is that you will need to use games specifiaclly made or updated to support this hardware. These games are actualy marked as Xbox One X Enhanced. Without this the game will run as normal, and you may see some performance enhacements but nothing that will be too dramatic.

Playstation 4

  • Released: November 15, 2013
  • Developer: Sony
  • Most powerful console: PS4 Pro
  • CPU: Semi-Custom 8-core AMD x86-64 Jaguar 1.6Ghz CPU
  • Memory: 8GB GDDR5
  • Graphics: Semi-Custom AMD GCN Radeon
  • GFLOPS: 4.2 Teraflops

The PS4 Pro is not the most powerful but it is still up there. This console still produces 4K output in PS4 Pro Enhanced games with better textures that take advantage of this extra power. But if power is not everything for you the PS4 still has a lot of other things it can give you over the Xbox One.
